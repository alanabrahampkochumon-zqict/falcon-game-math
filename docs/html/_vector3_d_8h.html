Configuration
Define ENABLE_FGM_SHADER_OPERATORS to enable comparison operators (>, <, etc.). Even if disabled, functional comparisons like greaterThan() remain available. Define FORCE_SCALAR to turn off SIMD which is on by default on supported hardware.
