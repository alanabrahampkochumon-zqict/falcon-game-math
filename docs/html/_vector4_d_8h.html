Detailed Description

Templated 4D Vector supporting integral, floating-point, and boolean types.

Author
Alan Abraham P Kochumon
Date
Created on: January 26, 2026
Todo
Provides a high-performance vector implementation with SIMD acceleration and support for component-wise operations.
Template Parameters
TType of Vector4D components. Must satisfy Arithmetic.
Note
Arithmetic operations are restricted to numeric types via StrictArithmetic.
Configuration
  • Define ENABLE_FGM_SHADER_OPERATORS to enable comparison operators (,, <, etc.). Even if disabled, functional comparisons like gt remain available.
  • Define FORCE_SCALAR to disable SIMD optimizations (enabled by default on supported hardware).

Scale the vector by a scalar value. Multiplies scalar by each component of the vector and returns a new vector.

Note
