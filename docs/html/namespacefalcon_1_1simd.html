std::size_t maxAlignAs )
constexpr

A calculator function that gives Aligned Byte Size, Padding, Packed Register Width, and Register Count when provided with an unaligned byteSize and maximumAlignment possible.

Parameters
totalByteSizeTotal unaligned byte size.
maxAlignAsMax possible alignment for a given CPU architecture. 16 for SSE, 32 for AVX and AVX2, and 64 for AVX512
Returns
PackedSize containing alignedByteSize, padding, register width and register count.
