auto fgm::operator* ( S scalar,
const Vector4D< T > & vector ) -> Vector4D< std::common_type_t< T, S > >

Scale the vector by a scalar value. Multiplies scalar by each component of the vector and returns a new vector.

Note
Promotes the result to the std::common_type_t of T and S.
Operation is restricted to numeric types via StrictArithmetic.
Template Parameters
SNumeric type of the scalar. Must satisfy StrictArithmetic.
Parameters
[in]scalarThe value to scale by.
[in]vectorThe vector to scale[RHS].
Returns
A new Vector4D scaled by scalar.
